HTML (HyperText Markup Language)
A linguagem de marcação usada para estruturar o conteúdo de uma página web.
CSS (Cascading Style Sheets)
Utilizado para estilizar e formatar elementos HTML, tornando-os visualmente atraentes.
JavaScript
Uma linguagem de programação que permite adicionar interatividade e funcionalidade dinâmica às páginas web.
React
React é uma biblioteca JavaScript para criar interfaces de usuário interativas.
SASS
SASS é um pré-processador CSS que facilita a escrita e organização de estilos.
