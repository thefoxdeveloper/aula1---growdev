Aprendendo HTML

Lista Ordenada:

  1. HTML: Linguagem de Marcação de Hipertexto
  2. Tags: Elementos utilizados para estruturar o conteúdo HTML
  3. Elementos: Partes individuais de uma página HTML

Lista Não Ordenada:

  • Web Browser: Software usado para visualizar páginas web
  • Hyperlink: Um link que permite a navegação entre páginas
  • HTML5: A versão mais recente do HTML

Lista de Descrição:

HTML
Linguagem de marcação usada para criar páginas web.
Tag
Um elemento usado para marcar partes específicas do conteúdo HTML.
Elemento
Uma parte individual de uma página web, como um cabeçalho ou um parágrafo.
